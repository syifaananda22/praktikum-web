# 🌐 Praktikum Web - Exercise 1 & 2

Repository ini merupakan hasil pengerjaan **Praktikum Mata Kuliah Pemrograman Web** yang mencakup Exercise 1 dan Exercise 2.  
Project ini dibuat sebagai implementasi dari materi dasar pengembangan website menggunakan HTML dan Bootstrap.

---

## 📌 Deskripsi Praktikum Web

Praktikum Web ini bertujuan untuk melatih mahasiswa dalam:
- Memahami struktur dasar HTML
- Membuat halaman web statis
- Menghubungkan beberapa halaman (multi-page)
- Menggunakan framework CSS (Bootstrap)
- Mendesain tampilan website sederhana namun rapi

Dalam praktikum ini, dibuat sebuah website yang berisi halaman profil masing-masing anggota kelompok yang saling terhubung melalui navigasi.
